# AgileScope

Project management tool for teams to organize projects, tasks, and sprints.

## Tech stack

Client:

- React
- TypeScript
- Gatsby.js
- Context API
- React Query
- React Hook Form
- Chakra UI

Server:

- Node.js
- Express
- TypeScript
- MongoDB
- Mongoose
- Nodemailer
- Sendinblue
- Speakeasy 2FA
